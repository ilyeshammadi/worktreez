# Git worktreez


## Create
```sh
$ worktreez create \
    --branch_name FEATURE-X \
    --repos_path ~/repos \
    --dest_path ~/repos/dest \
    --repo_name repo1 \
    --repo_name repo2 \
    --repo_name repo3 \
```

## Delete
```sh
$ worktreez delete \
    --branch_name FEATURE-X \
    --dest_path ~/repos/dest \
```
